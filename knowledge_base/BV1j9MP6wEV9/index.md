## 内容提要
本视频是一篇面向零基础用户的桌面Agent入门教程，以国产工具 **KimiWalk** 为核心案例，系统演示了桌面Agent的八大能力：本地文件读写（整理Excel、生成PPT）、浏览器操控（通过Wave Bridge插件自主搜索社交媒体信息）、Agent集群（多Agent并行处理海量数据如电商复盘）、Go模式（数小时无人值守深度工作）、插件生态（飞书、Canva、Computer Use）、Skill技能（PPT美化、投研报告生成）、Agent MD行为规范文件、以及定时任务。视频采用实际日常办公案例——帮表妹分析高考志愿填报学校——串联了从Excel数据清洗到PPT生成再到社交媒体口碑调研的完整流程，让零基础用户也能快速上手。

## 核心要点
- 桌面Agent运行在本地电脑上，能直接读写操作本地文件和操控浏览器，与网页版AI有本质区别
- KimiWalk 安装简单：官网下载 → 扫码登录 → 选择 Walk 模式（非 Chat 模式）
- 本地文件操作：能读取Excel、筛选排序、上色标注、基于数据生成可编辑PPT
- 浏览器操控：安装 Wave Bridge 插件后，Agent 可用自己登录态的浏览器自主搜索、点击、采集社交媒体信息
- Agent集群：最多可同时拉起300个Agent分头处理不同数据维度，适合电商复盘等多数据源场景（需99元/月会员）
- Go模式：设定明确目标和验收标准后，Agent可持续数小时自主工作，自行检查、返工、循环直至完成
- 插件系统：飞书（多维表格+仪表盘+发送消息）、Canva可画（战报海报）、Computer Use（通用桌面操控）
- Skill技能：归藏PPT（美化PPT样式）、Apple Reset（投研报告生成），可安装官方或自定义技能
- Agent MD文件：项目文件夹中的Markdown行为规范，会在Agent执行任何操作前注入为系统提示词，可搭建长期记忆系统
- 定时任务：可设置每天定时自动执行，如读取飞书日历生成日程看板、监控机票价格并推送提醒
- 目前KimiWalk仍为Beta版，做不到100%交付，使用时需要一定运气和人工验收

## 00:42-02:58 第一大能力：本地文件读写
- KimiWalk安装：Kimi官网 → 下载安装 → 扫码登录
- 两个模式：Chat（等同于网页版Kimi）和 Walk（真正的桌面Agent）
- Walk 模式核心能力：读写操作本地文件
- 实战案例：一份混乱的985/211近三年各省份分数线Excel表格
  - 任务：筛选广东省某分数段内可报考学校，按分数线从高到低排列
  - 开启「全部允许」权限后，Agent直接在WPS中完成筛选、排序、上色
  - 进阶任务：基于筛选结果生成志愿填报PPT，Agent自动读取表格、新建PPT、制作可视化图表
- 还能操作命令行、下载软件、执行脚本、压缩图片/视频、转换文件格式

## 02:58-04:19 第二大能力：浏览器操控
- 安装 Wave Bridge 浏览器插件后，Agent可操控浏览器
- 使用用户自己带登录态的浏览器，能访问需要登录的社交媒体和不开放API的网站评论
- 实战案例：对筛选出的学校，让Agent自主搜索社交媒体上的王牌专业、保研就业、宿舍口碑等真实评价
- Agent自主打开浏览器、搜索、点击，全程自动操作
- 用户在此期间用浏览器做其他事也不受影响
- 还能批量操作：如一口气关注100个博主

## 04:19-06:28 第三大能力：Agent集群
- 当资料多、数据量大时，单个Agent效率不足，可使用Agent集群
- Kimi特色：最多同时拉起300个Agent分头干活
- 实战案例：618电商大促复盘——文件夹含订单、投放、退款、客服、物流、财务数据及会议纪要
  - 多个不同专长的Agent分头读取不同文件
  - 最终整合输出：数据复盘报告 + 汇报PPT + 老板可能追问的QA
  - 报告涵盖：总结摘要、整体达成情况、投放分析、退款原因分析等
  - PPT图文并茂，关键数据自动做成图表
- 也适用于苹果产品策略分析、写字楼弱电方案等
- 需99元/月以上会员

## 06:28-08:14 第四大能力：Go模式（深度自主工作）
- 普通模式下Agent几分钟完成一个小步骤，等人类反馈
- Go模式：给明确任务目标和验收标准后，Agent可持续数小时自主工作
- Agent自主：做计划 → 动手做 → 检查结果 → 返工 → 循环 → 直到完成
- 实战案例：做大项目前讨论清楚需求，形成PRD和验证标准，开启Go模式让Agent逐条对照完成
- 适合「睡前下一个Go，早上起来验收」
- 注意事项：额度消耗较快

## 08:14-09:42 第五大能力：插件系统
- 飞书插件：将复盘数据整理为带详情数据看板的多维表格，直接发送给老板（拆成十几个表和视图，仪表盘可视化）
- 可画(Canva)插件：一句话将项目复盘数据做成战报海报，在Canva中可继续编辑
- Computer Use：比浏览器操控更通用，可直接在桌面上做精细化操作、操控其他软件

## 09:42-10:31 第六大能力：Skill技能
- 技能文件夹：可上传常用本地Skill
- 官方推荐技能：
  - 归藏PPT：将普通PPT优化为风格强烈、排版精致的版本
  - Apple Reset：输出机构级投研报告，质量非常高

## 10:31-11:42 第七大能力：Agent MD（行为规范文件）
- Agent MD是一份Markdown文档，内容会在Agent做任何事之前注入为系统提示词
- 可在项目文件夹中手动创建，也可让KimiWalk一键生成
- 可搭建有长期记忆的系统
- UP主提供小型模板：让Agent在复杂任务下先输出PRD确认、不盲目肯定用户、随时记录决策和记忆、有序组织项目文件

## 11:42-12:11 第八大能力：定时任务
- 设置定时任务让Agent到点自动开工
- 示例：每天早上8点读取飞书日历生成今日日程看板
- 示例：每天查询国庆机票价格，低于790元时发飞书消息提醒

## 12:11-13:13 总结：八大能力回顾
1. 操作命令行、读写本地文件
2. 操控浏览器、甚至操控电脑
3. Agent集群（拆成多个Agent并行干活）
4. Go模式（给定目标，自主干到底）
5. 接入特定插件
6. 使用专门技能
7. Agent MD立规矩、搭系统
8. 定时任务
- KimiWalk目前是Beta版，做不到100%交付，需要运气+人工验收

## 行动建议/学习建议
1. 下载KimiWalk（Kimi官网 → 找到KimiWalk → 安装 → 扫码登录），先试用Walk模式的本地文件操作
2. 准备一个日常工作中的真实Excel表格，让Agent尝试整理、筛选、生成报告，体验本地文件读写能力
3. 安装Wave Bridge浏览器插件，尝试让Agent帮你搜索某个主题的社交媒体评价
4. 如果每月预算允许（99元），开通会员体验Agent集群功能处理多维度数据任务
5. 尝试Go模式前先花时间写好PRD和验收标准文档，这是成功的关键
6. 将常用的Agent行为规范写成Agent MD文件放入项目文件夹，建立标准化工作流程
7. 设置一个简单的定时任务（如每日新闻摘要），体验自动化日常工作
8. 注意：KimiWalk为Beta版，目前偶尔会有交付不完整的情况，重要任务务必人工验收
