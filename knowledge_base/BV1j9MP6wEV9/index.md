## 内容提要
本视频是一篇面向零基础用户的国产桌面Agent——KimiWalk 的入门到进阶教程。UP主秋芝2046以日常办公场景为线索，系统演示了 KimiWalk 的八大核心能力：本地文件读写、浏览器操控、Agent集群、Go模式（长时自主执行）、插件接入、Skill技能、Agent MD系统提示词、以及定时任务。视频使用高考志愿填报、618电商复盘、飞书多维表格自动生成等真实案例，展示了桌面Agent如何从简单的"AI聊天"进化为能独立操作电脑、自主完成复杂任务的数字员工。教程强调：桌面Agent运行在用户自己的电脑上，具备本机文件系统和浏览器（保持登录状态）的完整访问能力，适合白领办公场景下的自动化提效。

## 核心要点
- **能力一：操作命令行与读写本地文件** — KimiWalk 可直接读取、修改本地文件（Excel/PPT/图片/视频），支持压缩、格式转换、执行脚本等操作。
- **能力二：浏览器操控（Wave Bridge）** — 通过浏览器插件操控用户自己的浏览器（保持登录状态），可访问需要登录的社交媒体、抓取不开放API的网站数据，且操控期间不影响用户正常使用浏览器。
- **能力三：Agent集群** — 最多可同时拉起300个Agent分工协作，适合数据多、维度广、产出需求多样的场景（如618电商复盘、苹果全产品线策略分析）。
- **能力四：Go模式（长时自主执行）** — 给Agent一个明确目标+验收标准，它可自行循环"计划→执行→检查→返工"数小时甚至十几小时，用户睡醒即可验收结果。
- **能力五：插件接入** — 已集成飞书、钉钉、可画(Canva)、WPS等常用工具插件；另有 Computer Use 插件可操控电脑桌面上的任意软件。
- **能力六：Skill技能** — 可安装官方或自定义Skill来提升特定任务质量，如PPT美化Skill、Apple Research深度研报Skill。
- **能力七：Agent MD（系统提示词）** — 在项目文件夹中编写Markdown规则文件，注入为Agent的系统提示词，可约束行为规范、建立长期记忆机制。
- **能力八：定时任务** — 设定Agent按时间表自动执行任务，如每日早8点读取飞书日历生成日程看板、监控机票价格并在低于阈值时推送提醒。

## 按时间/段落整理
### 00:00-00:42 开场：为什么要关注桌面Agent
UP主指出2026年Agent是绕不开的话题，但90%的用户仍停留在AI聊天阶段。之前出过海外Codex教程，但很多朋友卡在注册，所以本次改用国产桌面Agent KimiWalk。

### 00:42-01:18 安装与两种模式
在Kimi官网下载KimiWalk桌面应用，扫码登录。打开后有两种模式：Chat模式（等同于网页版Kimi）和Walk模式（真正的桌面Agent）。

### 01:18-02:58 能力一：本地文件操作（案例：高考志愿填报）
- 新建项目 → 选择本地文件夹作为Agent工作区
- 案例：文件夹中有985/211近三年各省份分数线Excel表格，年份省份学科混在一起
- Agent按要求筛选特定分数段、特定省份的学校，按分数线从高到低排序，自动上色
- 随后Agent基于筛选结果自动新建PPT，包含可视化图表，可直接编辑

### 02:58-04:27 能力二：浏览器操控（Wave Bridge）
- 安装Wave Bridge浏览器插件后，Agent可操控用户自己的浏览器（保持登录状态）
- 案例：Agent自动搜索社交媒体上的学校口碑、王牌专业、保研率、宿舍评价等信息，更新到PPT中
- 操控过程中用户可正常使用浏览器，互不影响
- 也支持批量操作，如"一口气关注100个博主"

### 04:27-06:27 能力三：Agent集群
- 最多可同时拉起300个Agent分工协作，需99元/月以上会员
- 案例一：618电商复盘 — 订单/投放/退款/客服/物流/台账数据 + 会议纪要 + 老板反馈 → 自动生成数据复盘报告 + 汇报PPT + QA问答
- 案例二：苹果所有年份产品策略分析 — 自动查询、提炼策略逻辑、多维度展开、划分战略阶段
- 案例三：整套公司弱电方案 — 仅提供平面图，Agent自动生成包含自绘图表、详尽考虑的完整方案

### 06:27-08:14 能力四：Go模式（长时自主执行）
- 普通模式：Agent几分钟完成一个小步骤就等人类反馈
- Go模式：给Agent明确任务终点+验收标准，它自动循环"计划→执行→检查→返工"，可持续数小时甚至十几小时
- 案例：将PRD文档放入项目，开启Go模式，Agent逐条对照需求完成并自我检验；用户睡一觉回来验收即可
- 适合"睡前下一个Goal，早上起来验收"的场景，但额度消耗快

### 08:14-09:38 能力五：插件接入
- 飞书插件：Agent将618复盘数据拆成十几个表和视图，生成带可视化看板的飞书多维表格并直接发给老板
- 可画(Canva)插件：一句话将项目复盘数据生成战报海报，点击链接即可在可画中继续修改
- Computer Use插件：比浏览器操控更通用，可直接在电脑桌面上做系统级操作、操控任意软件

### 09:38-10:29 能力六：Skill技能
- 位于插件菜单左上角，可安装官方或上传自定义Skill
- 推荐Skill：PPT美化（一键将普通PPT变为风格强烈、排版精致的样式）、Apple Research（输出机构级深度研报）
- Skill解决的是Agent在特定任务上"做得好不好"的问题

### 10:29-11:34 能力七：Agent MD（系统提示词）
- Agent MD是一份Markdown文档，在Agent做任何事之前优先注入为系统提示词
- 可在项目文件夹中手动创建或让KimiWalk一键生成
- 可基于Agent MD机制搭建长期记忆系统
- UP主提供小模板：让Agent在复杂任务下先输出PRD确认、不盲目肯定用户、随时记录决策和记忆、有序组织项目文件

### 11:34-12:56 能力八：定时任务
- 在定时任务中新建，输入任务和需求即可
- 案例一：每天早上8点读取飞书日历，生成今日日程看板
- 案例二：每天查询国庆机票价格，低于790就发飞书消息提醒

### 12:56-13:10 总结与展望
回顾八大能力，鼓励观众把桌面Agent当成工作伙伴。提示KimiWalk仍是Beta版，还不能做到"甩手掌柜"百分百交付，有一定运气成分。

## 行动建议/学习建议
1. **立即上手**：前往Kimi官网下载KimiWalk桌面应用，扫码登录后从"新建项目"开始体验本地文件操作。
2. **安装Wave Bridge插件**：激活浏览器操控能力，这是打通线上数据与本地文件的关键桥梁。
3. **从简单任务起步**：先用单Agent做Excel整理、PPT生成等小任务，熟悉后再尝试Agent集群和Go模式。
4. **搭建Agent MD规范**：为自己的常用项目编写Agent MD模板（含PRD确认流程、行为约束、记忆记录规则），提升Agent协作质量。
5. **善用定时任务**：将日报/周报数据采集、价格监控等重复性工作交给定时任务，实现"无感自动化"。
6. **注意版本限制**：Agent集群需99元/月以上会员；Go模式额度消耗较快，建议在明确验收标准后再开启。
